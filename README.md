# Email Generator AI (Gemini v1)

This application generates professional Azure quota increase request emails based on rough drafts. It features a unified, enterprise-grade table UI standard that is fully compatible with Microsoft Word and Outlook.

## Features
- **Strict Grid Table UI**: Professional, high-contrast tables with white backgrounds and black borders.
- **Word/Outlook Optimized**: Inline styles ensure formatting is preserved when copied to email clients.
- **Multi-lingual Support**: EN and PT proofreading and translation capabilities.
- **Auto-Deployment**: Integrated GitHub Actions workflow for static hosting.

## Tech Stack
- React + TypeScript
- Vite
- GitHub Actions (CI/CD)

## Getting Started
1. Install dependencies: `npm install`
2. Run development server: `npm run dev`
3. Build for production: `npm run build`
